--Workspaces
--They are stored in cui files created by an Autocad customer.
--After creation they can be loaded directly into a larger cui file and will become part of that file.
--When loaded as partial customizations they retain the ability to update inside the larger file when changes to that individual, smaller cui file occur.
--They are good to save for editing later and best kept out of the way until then.
